# Customer-Shopping-Behavior-Analysis

---

## Overview

This project analyzes consumer shopping behavior for a retail business to uncover purchasing trends, customer preferences, and key drivers of repeat purchases. The goal is to support data-driven decisions that improve customer engagement, optimize marketing strategies, and enhance product performance. The project follows an end-to-end analytics workflow using Python, SQL, and Power BI.

---

## Dataset

The dataset contains customer-level shopping information, including demographic attributes, product categories, purchase behavior, discounts, payment preferences, and review data.
Python was used to clean and prepare the raw data before analysis to ensure consistency, accuracy, and usability across tools.

---

## Tools & Technologies

* **Python**: Data loading, exploratory data analysis (EDA), and data cleaning
* **PostgreSQL**: Structured data analysis using SQL queries
* **Power BI**: Interactive dashboard creation and data visualization
* **Gamma**: Presentation (PPT) creation for business storytelling
* **GitHub**: Version control and project documentation

---

## Analysis Workflow

1. Loaded and explored the dataset using Python
2. Performed data cleaning and preprocessing (handling missing values, formatting, validation)
3. Stored cleaned data and ran SQL queries on a PostgreSQL server
4. Extracted insights on customer segments, purchasing patterns, and loyalty drivers
5. Built an interactive Power BI dashboard to visualize trends and KPIs
6. Summarized insights in a written report and presentation

---

## Dashboard

The Power BI dashboard highlights:

* Customer segmentation and purchasing behavior
* Category-wise and demographic-based trends
* Factors influencing repeat purchases and customer loyalty
* Key metrics to support marketing and product strategy decisions

---

## Results & Insights

* Identified key customer segments contributing to higher revenue
* Highlighted product categories and discounts driving repeat purchases
* Revealed behavioral patterns across demographics and sales channels
* Provided actionable insights for improving customer engagement and marketing effectiveness

---

## How to Run

1. Clone the repository
2. Run the Python script or notebook to explore and clean the dataset
3. Load the cleaned data into PostgreSQL and execute the SQL queries
4. Open the Power BI `.pbix` file to view the interactive dashboard
5. Review the report and presentation for summarized insights
